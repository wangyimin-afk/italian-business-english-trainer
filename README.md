# Italian Business English Trainer

一个静态 Web App，用来训练“意大利母语者说商务英语”的听力和会议反应能力，重点是读、背、熟悉意大利式商务英语表达节奏。

## 功能

- 内置 120 句意大利商务英语背诵句库
- 按会议开场、确认理解、数字成本、采购供应、交期风险、变更问题、谈判表达等分类筛选
- 复制全部句库用于离线背诵
- 不内置真实机构、真实客户、真实项目或敏感会议内容

## 隐私处理

仓库内不包含真实机构、项目、地点或人名。

## 本地打开

直接打开 `index.html` 即可。

## GitHub Pages

本项目是纯静态页面，可以直接部署到 GitHub Pages。
