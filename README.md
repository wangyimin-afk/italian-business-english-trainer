# Italian Business English Trainer

一个静态 Web App，用来训练“意大利母语者说商务英语”的听力和会议反应能力，重点覆盖商务会议里的结论、数字、责任、时间、原因和下一步。

## 功能

- 粘贴会议转写文本
- 内置 120 句意大利商务英语背诵句库
- 按会议开场、确认理解、数字成本、采购供应、交期风险、变更问题、谈判表达等分类筛选
- 复制全部句库用于离线背诵
- 手动输入敏感词并替换为通用占位词
- 提取商务会议关键词
- 抓取数字、比例、时间和数量
- 识别常见意大利式商务英语口头禅
- 生成业务主线、三遍听力训练步骤和会议确认句
- 导出 Markdown 训练笔记
- 不内置真实机构、真实客户、真实项目或敏感会议内容

## 隐私处理

仓库内不包含真实机构、项目、地点或人名。真实敏感词应在网页里临时输入，App 会在浏览器端替换，不会上传到服务器。

## 本地打开

直接打开 `index.html` 即可。
