# Overseas Trade Language Library

一个静态 Web App，用来训练海外贸易工作中的语言反应能力。它不是传统语言课程，而是面向真实客户沟通的工作语言库。

## 功能

- 120 句意大利式商务英语背诵句库
- 50 个海外贸易核心词，覆盖英语、意大利语、印度英语、马来语、韩语、日语、芬兰语、德语
- 多语言词条点击朗读
- 日语五十音、韩语基础音节、意大利语、芬兰语、德语基础发音格子
- 意大利、印度、马来西亚/东南亚、韩国、德国、芬兰客户说英语的方式
- 不内置真实机构、真实客户、真实项目或敏感会议内容

## 隐私处理

仓库内不包含真实机构、项目、地点或人名。

## 朗读说明

朗读使用浏览器自带 `speechSynthesis`。不同系统和浏览器可用语音不同，因此某些语言可能会用近似语音朗读。
