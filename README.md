# Italian Business English Trainer

一个静态 Web App，用来训练“意大利母语者说商务英语”的听力，重点覆盖采购、供应链、成本、模具、双供应源、海运和需求波动等会议场景。

## 功能

- 粘贴会议转写文本
- 手动输入敏感词并替换为通用占位词
- 提取采购/供应链关键词
- 抓取数字、比例、时间和数量
- 识别常见意大利式商务英语口头禅
- 生成业务主线、三遍听力训练步骤和会议确认句
- 导出 Markdown 训练笔记

## 隐私处理

仓库内不包含真实供应商、客户、工厂、项目或人名。真实敏感词应在网页里临时输入，App 会在浏览器端替换，不会上传到服务器。

## 本地打开

直接打开 `index.html` 即可。
